<양꼬치>

읽기 단계

- 입력: 무엇이 주어지나?
  양꼬치 n과 음료수 k, 양꼬치 (n) =10인분 주문시 음료수 (k) = 1개는 서비스
- 출력 : 무엇을 반환해야 하나?
- 제한 : 범위, 예외는?
- 예시 : 입력 -> 출력 이해했나?

[제미나이 answer]

- 입력: 양꼬치 주문 수 $n$, 전체 마신 음료수 개수 $k$
- 출력: 지불해야 할 총 금액
- 제한: $0 < n < 1,000$ (양꼬치는 최소 1인분 이상 주문)$n / 10 \le k < 1,000$ (마신 음료수 $k$는 서비스로 받은 개수보다 항상 같거나 많음)
- 예시 이해: $n=10, k=3$일 때: 양꼬치 10인분이므로 서비스 음료 1개 발생. 실제 돈 내는 음료는 $3 - 1 = 2$개.총액: $(12,000 \times 10) + (2,000 \times 2) = 124,000$원.

계획 단계

1.  음료수는 k는 n의 비례해서 먹지 않으며, 서비스 음료를 포함해 계산해야 한다.

2.  k는 서비스 음료와 주문 음료 비례해서 계산한다.

3.  if와 for 활용

[제미나이 answer]

1. 서비스 음료 개수 파악: 양꼬치 $n$을 10으로 나눈 몫이 서비스 음료의 개수입니다.예: 64인분 주문 시 $64 \div 10 = 6.4$ 이므로 6개가 서비스입니다. (정수 부분만 취함)

3. 총액 계산: (양꼬치 가격 $\times n$) + (음료수 가격 $\times$ 지불할 음료 개수)

[제미나이 answer]

    function solution(n, k) {
    const serviceDrink = Math.floor(n / 10);

    const paidDrink = k - serviceDrink;


    const lambPrice = 12000;
    const drinkPrice = 2000;


    const answer = (n * lambPrice) + (paidDrink * drinkPrice);

    return answer;

}

[코드 최적화]

function solution(n, k) {
return (n _ 12000) + ((k - Math.floor(n / 10)) _ 2000);
}

검증 단계

[제미나이 answer]

예제 1 ($n=10, k=3$):서비스: $10 / 10 = 1$개유료 음료: $3 - 1 = 2$개계산: $(12,000 \times 10) + (2,000 \times 2) = 124,000$ (통과)

예제 2 ($n=64, k=6$):서비스: $64 / 10 = 6.4 \rightarrow 6$개유료 음료: $6 - 6 = 0$개계산: $(12,000 \times 64) + (2,000 \times 0) = 768,000$ (통과)

경계값 ($n=1, k=1$):서비스: $1 / 10 = 0.1 \rightarrow 0$개계산: $(12,000 \times 1) + (2,000 \times 1) = 14,000$ (정상)
